# Dominion

**DOM**ain **I**dentification for **N**etworks of **I**mmunolabeled **O**bject **N**eighborhoods.

A workflow for quantitatively segmenting and tessellating astrocyte domains from GFAP and DAPI immunofluorescence images.

## Goal

Given two-channel fluorescence imagery — GFAP (astrocyte processes) and DAPI (nuclei) — produce a per-cell partition of tissue space that approximates each astrocyte's territorial domain, and extract quantitative descriptors of those domains (area, shape, neighbor relationships, overlap, etc.).
